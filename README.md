# Wordle Japanese Bot

A bot for [Wordle Japanese](https://wordle.mottox2.com/).

## Use

```bash
git clone https://github.com/Tozaburo/wordle-japanese-bot
cd wordle-japanese-bot
npm install
npm run dev
```

## Features

In default, the `main()` function runs.
There are also `test()` function to test the bot.
